# Seed Repository - Comprehensive Summary

### Project Overview

**Seed** is a **Dynamic Development Environment Setup** system that automatically detects your computing context (VPS, Codespaces, local dev, CI/CD, container) and intelligently installs only the tools appropriate for that environment.

**Repository**: https://github.com/ebowwa/seed

---

### Key Components

| Component | File | Purpose |
|-----------|------|---------|
| **Main Setup** | `setup.sh` (1,675 lines) | Environment detection & tool installation |
| **Configuration** | `situations.yaml` (319 lines) | Declarative environment/tool definitions |
| **Chat Wrapper** | `chat.sh` | Persistent Claude Code conversations with Doppler |
| **Claude Config** | `.claude/` | Claude Code settings, commands, MCP servers |
| **Documentation** | `docs/` | 12+ comprehensive guides |

---

### Architecture

```
Environment Detection → situations.yaml → Tool Installation
                                     ↓
           Claude Code (settings + MCP servers + skills)
                                     ↓
        chat.sh (persistent conversations via Doppler)
```

### Chat Script (chat.sh)

Persistent conversation wrapper that:
- Loads secrets from Doppler (configurable project/config)
- Maintains conversation history in `/tmp/c.txt`
- Sends full context to remote Claude each time
- Returns response with all conversation memory

Usage:
```bash
./chat.sh "your prompt"
./chat.sh "prompt" --project myproj --config dev
```

---

### Distributed Claude

The `distributed-claude` skill enables communicating with a remote Claude instance on the Hetzner server. This allows:
- Different backends (Z.ai GLM vs Anthropic)
- Independent context/persistence
- Multi-Claude collaboration

```bash
ssh ebowwa-deptwar "cd ~/seed && ./chat.sh 'prompt'"
```
