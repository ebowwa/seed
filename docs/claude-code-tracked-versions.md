## How to Track New Changes

1. **Automatic**: GitHub Actions runs daily at 9:00 AM UTC
2. **Manual**: Run `/track-claude-code` slash command in Claude Code
3. **Manual Trigger**: Go to Actions → Track Claude Code Changes → Run workflow
